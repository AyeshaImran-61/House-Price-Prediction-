#  House Price Prediction – Data Preprocessing

A comprehensive data preprocessing project for house price prediction that prepares raw real estate data for machine learning models. The notebook demonstrates essential data cleaning, preprocessing, feature engineering, and exploratory data analysis (EDA) techniques to improve data quality and model performance.

---

##  Project Overview

The objective of this project is to transform raw housing data into a clean and structured dataset suitable for predictive modeling. The preprocessing pipeline includes handling missing values, removing duplicates, detecting outliers, encoding categorical features, scaling numerical variables, and performing exploratory data analysis to understand relationships between housing attributes and prices.
